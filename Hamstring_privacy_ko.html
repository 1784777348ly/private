스포츠 성능

마지막 업데이트: 2025년 11월 28일

소개

"스포츠 성능" 앱에 오신 것을 환영합니다. 우리는 개인정보의 중요성을 이해하고 귀하의 프라이버시를 보호하기 위해 최선을 다하겠습니다.

권한 사용

블루투스 권한

스포츠 센서, 심박수 밴드 등 장치를 연결하고 스포츠 데이터를 수집하는 데 사용됩니다.

사진 라이브러리 권한

스포츠 분석 스크린샷, 훈련 기록 등을 저장하고 읽는 데 사용됩니다.

카메라 권한

스포츠 비디오를 녹화하여 동작 분석과 자세 교정을 수행하는 데 사용됩니다.

로컬 파일 권한

스포츠 데이터 보고서, 훈련 계획 등의 파일을 저장하는 데 사용됩니다.

네트워크 권한

당신의 스포츠 데이터를 클라우드에 동기화하고 최신 훈련 콘텐츠를 얻는 데 사용됩니다.

수집하는 정보

  • 장치 정보: 장치 모델, 운영 체제 버전
  • 스포츠 데이터: 심박수, 운동 시간, 소비 칼로리
  • 미디어 파일: 스포츠 사진, 비디오 (허가한 경우에만)
  • 개인 데이터: 키, 체중, 나이 (개인화 계산을 위해)
  • 사용 데이터: 기능 사용 빈도, 훈련 시간

정보 사용 방식

  • 개인화된 스포츠 분석 보고서 제공
  • 앱 성능과 안정성 최적화
  • 스포츠 보고서와 진보 추세 생성
  • 고객 지원 서비스 제공

데이터 저장과 보안

  • 모든 데이터 전송은 SSL/TLS 암호화를 사용합니다
  • 개인정보 접근 권한을 엄격히 제한합니다
  • 사용 목적에 필요한 기간만 보존합니다

귀하의 권리

  • 접근권: 귀하의 개인정보를 조회합니다
  • 정정권: 부정확한 개인정보를 수정합니다
  • 삭제권: 귀하의 개인정보 삭제를 요청합니다
  • 동의 철회권: 언제든지 허가된 정보 수집을 철회합니다
